**DutySpeed** is a lightweight, automated speedrun timer for Final Fantasy XIV. It helps you track your performance in dungeons, trials, and raids without needing to manually start or stop a stopwatch.

![DutySpeed Icon](https://raw.githubusercontent.com/YourUsername/DutySpeed/main/icon.png)

## ‚ú® Features
* **Automatic Detection**: Starts the timer the moment the "Duty Commenced" message appears.
* **Smart Splits**: Automatically records time stamps for every boss defeat.
* **Top 5 History**: Maintains a leaderboard for every unique duty you run.
* **Party Tracking**: Records who you ran with and what jobs they played (hover over history to view).
* **Auto-Open**: Optionally pops the window open only when you enter an instance.
* **Record Browser**: View your personal bests for any dungeon even while standing in town.

### Custom Repository (Testing)
To try the latest features before they hit the main repo:
1. Open Dalamud Settings by typing `/xlsettings` in-game.
2. Navigate to the **Plugin Installer** > **Settings** tab.
3. Add the following URL to the **Custom Plugin Repositories** list:
   `https://raw.githubusercontent.com/YourUsername/DutySpeed/main/repo.json`
4. Search for "DutySpeed" in the installer and click **Install**.
